Simple Python Version Management: pyenv

介绍

pyenv 让你更容易的在多个 Python 版本中切换。它使用非常简单，跟传统 UNIX 只关注一件事的工具一样。
这个项目从rbenv和ruby-build中 fork 出来，为了 python 修改
** pyenv 功能 **

  • 让你基于每个用户更改全局 Python 版本
  • 提供对么个项目的 python 版本
  • 允许你用环境变量重写 Python 版本
  • 一次可以搜索多个 Python 版本的命令。对于使用 tox 扩 python 版本测试非常用帮助

命令参考

Command Reference
跟 git 一样,pyenv 根据第一个参数代理成子命令
公共的子命令如下

  • pyenv commands
  • pyenv local
  • pyenv global
  • pyenv shell
  • pyenv install
  • pyenv uninstall
  • pyenv rehash
  • pyenv version
  • pyenv versions
  • pyenv which
  • pyenv whence
